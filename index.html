私について About

1986年生まれ、栃木県出身。
大学卒業後、自動車部品の製造業を10年間経験しました。
大学で情報処理を専攻していたため、元々プログラミングに興味があり、本業の合間をぬって独学でWeb制作を学習しました。
学習後はクラウドワークスおよびWebサイト制作会社から制作案件を受注し、主にコーディングを担当しておりました。フリーランスとして活動後、Webサイト制作会社に入社をしました。
制作会社ではコーディングだけでなく、バナーやランディングページのデザインなども担当しておりました。現在は会社員として働きながらWebサイトのコーディング業務を行なっています。

スキル Skill

プログラミングスキル

  • HTML

    適切なタグを使い分けることによって、スクリーンリーダーおよび検索ブラウザに考慮したコーディングが可能です。

  • CSS

    後々のサイト運用の事を考えて「予測できる」「再利用できる」「保守できる」「拡張できる」をモットーにコーディングします。CSS設計はBEMを採用します。

  • JavaScript(jQuery)

    アニメーションの実装が可能です。例えばハンバーガーメニューやアコーディオンメニューの開閉、スライダーの実装、お問い合わせフォームのバリデーションなど。

  • PHP

    条件分岐やループ処理、PHP関数を使用してWordPressのカスタマイズが可能です。

  • WordPress

    HTML&CSSを使用した静的サイトをカスタマイズして、オリジナルテーマの制作が可能です。一例としてカスタム投稿、サブループ、パンくずリストなどの実装が可能です。

  • レスポンシブ対応

    スマホやタブレットなど、どのデバイスで見てもレイアウト崩れがないように実装いたします。

ソフトウェアスキル

  • Photoshop

    写真の切り取り、色調の変更、テキストの追加、Photoshopからのコーディングなどが可能です。

  • Adobe XD

    Webサイトのデザインカンプの制作、デザインカンプからのコーディングなどが可能です。

  • Figma

    デザインカンプからのコーディングなどが可能です。
